# Chapter Practice

수업 중 진행한 C++ 장별 연습 문제와 개인 실습 코드를 정리한 폴더입니다.

## 구성 기준

- `01_basic/`: 기본 입출력, 명령행 인자, 예외 처리
- `02_dynamic_memory/`: 동적 메모리와 포인터 기반 배열
- `03_struct_class/`: 구조체와 클래스 기초
- `04_class_separation/`: 클래스 선언/구현 파일 분리
- `05_oop/`: 생성자, 소멸자, 동적 문자열 메모리, 상속/다형성
- `06_file_io/`: 파일 입출력 및 이진 파일 복사

미완성 파일은 `unfinished/` 폴더에 따로 보관합니다.
